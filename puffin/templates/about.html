Introduction

Puffin lets you simply run web applications, such as blog, forum, chat, file sharing, news reader and many others, which will be always reachable from any location or device.

Getting Started

First you need to login {% if config.SECURITY_REGISTERABLE %} or register {%- endif %}. Afterwards you can run any application from the catalog. Once started, your application will be available under your subdomain. If you have any questions, please ask on Forum.

Application Catalog

Puffin contains ever growing list of web applications. Adding new ones is easy thanks to popular Docker containerization technology - learn more how to add your own application.

Hosting

You can run applications on a managed platform - service is free and no setup is required. Alternatively you can choose to launch Puffin on your own server and let your friends run their applications on it.

Status

Keep in mind that this is a demo platform and all data and applications will be regularly deleted. Puffin software is still in alpha development stage, so it might be unstable - you can track the project progress on the Blog in the Code.
