User

In order to get started you need to Login or Register. Afterwards you can choose an application you'd like to use from the list and Install it. Once the application is ready you can click Open and begin configuring it. After you finish your application will be available on the internet - you can access and share it via it's address in browser address bar. Keep in mind that this is a demo platform and all data and applications will be regularly deleted. If you have any questions send an email to puffin@puffin.rocks or create an Issue on Github.

Developer

To submit your application simply fork our repository and send us a pull request - see CONTRIBUTING.md for details. You can also help in Puffin core development via Github. Puffin is written in Python using Flask, is based on Docker and stores data in PostgreSQL.
